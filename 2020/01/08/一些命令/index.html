一些命令

Windows

通过注册表开启RDP

REG ADD HKLM\SYSTEM\CurrentControlSet\Control\Terminal" "Server /v fDenyTSConnections /t REG_DWORD /d 00000000 /f

获取DNS版本

nslookup -q=txt -class=chaos version.bind 120.236.192.196

磁盘安全擦除分区（格式化）

format disk: /P:count

把D盘用随机数覆盖8次

Example: format D: /P:8

安全删除文件（无格式化）

cipher /w:absdirpath

对该盘下的未使用空间进行一次0x00，一次0xFF，一次随机数覆盖

Example:cipher /w:D:\test\

Linux

连接smb

smbclient //smb_server -U username

zip加密压缩

zip -rP password zip_filename source_dirname

rdp连接

rdesktop
